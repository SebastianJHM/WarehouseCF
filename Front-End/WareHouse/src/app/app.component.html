Instructions of use

1. Identify the problem that you want to solve, allocation or picking, and then go to the respective window in the Navigation Bar.

2. Enter the data file with format (.xlsx) based in the following templates: Template Data Allocation or Template Data Picking. To enter the file, press the button next to "Excel Data File" or drag and drop the file in the box below.

3. In the below part, where is the "Execution Parameters" introduce the limit time.

4. Press the "Send Data" button.

Note: If you do not accomplish the steps 2 or 3 the problem will not execute. You will get a better solution if you put a higher limit time.

5.1 For the "Allocation Problem", the solution will be presented in two windows. The first one will show the location of the reference in the corresponding rack and you can filter by reference. The other window show the number of the rack with the corresponding references, you can also filter by rack.

5.2 By the other part, the "Picking Problem" will show the windows with each picking route.

6. Finally, you can download in your computer the Allocation or Picking problem in the format (.xlsx).
